- Que es la keyword 'this'.
: Es una palabra reservada para representar a la clase en la que estamos situados al momento de escribirla

- Que es la abstraccion
: Es uno de los principios de la programacion orienada a objetos que nos permite limitar la informacion representada a la que nos importa en el caso de uso.

- Dif. entre clase abstracta y una interfaz
: La clase abstracta puede ser clase padre y tener clases hijas, la interfaz no. La clase abstracta utiliza extends, la interfaz utiliza implements. En la clase abstracta tenes tu firma y el desarrollo del metodo, la clase hija puede sobreescribir el metodo de la clase abstracta, en las interfaces tenes la firma pero no el desarrollo, la clase que implementa la interfaz esta obligada a sobreescribir sus metodos. Otra diferencia importante es que podes heredar todas las interfaces que quieras, pero solo se puede heredar una clase abstracta.

- Que es P.O.O.
: Es un paradigma de la programación que se basa en el uso de clases y objetos para representar la realidad, ayudando a reducir la cantidad de lineas de codigo.

- Dif. entre overloading y overriding
: Overloading se refiere a tener dos metodos con mismo nombre pero distintos parametros, los cuales se comportan de formas distintas, overriding es cuando tenemos dos metodos con mismo nombre y mismos parametros, pero uno de los metodos sobreescribe completamente el comportamiento del metodo anterior. 

- Que es el polimorfismo
: El polimorfismo es uno de los pilares de P.O.O que es la capacidad que tiene un metodo para comportarse de formas distintas en base a las circunstancias (overloading y overriding)

- Que es una interfaz
: Es una clase especial utilizada para definir el comportamiento de las clases, las clases que implementan una interfaz deben sobreescribir todos sus metodos con firma exacta.

- 4 pilares de P.O.O
: Polimorfismo, abstraccion, herencia y encapsulamiento.

- Que es una excepcion
: Es un evento que sucede en la ejecucion del programa que la interrumpe cuando sucede un error para lanzar un objeto con un mensaje acerca de lo que sucedió.

- Que es el java home
: Es la variable de entorno de java que indica la ubicacion del binario java

- Que es el m2 home o maven home
: Es la variable de entorno de maven que indica la ubicacion de los archivos .jar de maven
